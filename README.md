🚦 Traffic Light Simulation

A simple Traffic Light Simulation project built using HTML, CSS, and JavaScript.

✨ Features

- 🔴 Red Light
- 🟡 Yellow Light
- 🟢 Green Light
- 🔄 Automatic Light Switching
- 📱 Responsive Design
- 🎨 Simple and Clean UI

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📂 Project Structure

traffic-light-simulation/
│
├── index.html
├── style.css
├── script.js
└── README.md

🚀 How to Run

1. Clone or download this repository.
2. Open "index.html" in your browser.
3. Watch the traffic lights change automatically.

🎯 Learning Outcomes

This project helps beginners learn:

- HTML Structure
- CSS Styling
- JavaScript DOM Manipulation
- Timers and Intervals
- Basic Animation Concepts

📸 Preview

Traffic Light with automatic Red, Yellow, and Green signal transitions.
